# Installation

### What you'll need

- Administrator rights on a JIRA Cloud instance.
- At least one Agile Scrum board. (**NitroIQ will not work with Kanban boards.**)
- (Optional) Rum and Coke (because who doesn't like a Cuba Libre?)
